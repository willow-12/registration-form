# Registration Form (HTML, CSS, JS, PHP)

##  Description
This is a simple registration form project built using:
- HTML (structure)
- CSS (styling)
- JavaScript (validation)
- PHP (form processing)

##  Features
- User input form
- Client-side validation using JavaScript
- Server-side processing using PHP
- Displays submitted data

##  How to Run
1. Install XAMPP / WAMP
2. Place project folder in `htdocs`
3. Start Apache
4. Open browser:
   http://localhost/registration-form/

## Files
- index.html– Form UI
- style.css – Styling
- script.js – Validation
- process.php – Backend processing
